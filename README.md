# COC_puppetFiles

# Puppet Lab 9 – INET 4031

This repository contains Puppet manifest files created for Lab 9.

The project demonstrates:
- Creating and managing users and groups using Puppet
- Building a LAMP stack (Apache, PHP, MariaDB)
- Automating server setup using Puppet manifests
- Testing PHP using a phpinfo page

Files included:
- testing_puppet.pp: basic Puppet test file
- server_users_groups.pp: manages users and groups
- lamp_stack_server.pp: installs and configures LAMP stack
- phpinfo.php: PHP test file

This lab shows how infrastructure can be automated and managed using Puppet.
